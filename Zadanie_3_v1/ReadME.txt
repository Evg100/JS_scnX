Данный пример позволяет пользователям перемещать созданный вектор по карте.

Пользователь может нарисовать вектор с помощью панли рисования в правом верхнем углу 
Для перемещения вектора необходимо активировать соответствующую галочку под картой.

На карте может быть нарисовано несколько векторов сразу. Но перемещается каждый вектор отдельно.

Примечание: для перемещения линейного вектора лучше сначала выбрать опцию "navigate" на панели рисования
(значок руки).
